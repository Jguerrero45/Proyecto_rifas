A traves de este apartado podras agregar / eliminar numeros de la rifa segun la gente vaya pagando los boletos. Para agregar un boleto, simplemente introduce el numero del boleto y presiona el boton 'Enviar'

agregar como pagado: al agregar como pagado se elimina la opcion de comprar el numero mediante la pagina, es decir, el numero ya no estara disponible para la compra.

eliminar como pagado: al eliminar como pagado se habilita la opcion de comprar el numero mediante la pagina, es decir, vuelve a aparecer en la lista de numeros disponibles para la compra.

para apregar como pagado: seleccione pagado

para eliminar como pagado: seleccione no pagado
